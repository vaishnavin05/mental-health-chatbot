# Mental Health Companion Chatbot for Students

This project is an AI/ML-based mental health chatbot designed to support students by analyzing sentiment from text inputs and providing empathetic responses and coping strategies.

## Features
- Sentiment analysis using Machine Learning
- Dataset-based training
- Mood tracking visualization
- Student-focused mental health support
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- NLP (TF-IDF)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
